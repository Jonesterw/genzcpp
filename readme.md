Welcome to the GenZ C++ Repo

## Installation

1. Download `genzcpp.h`.
2. Put it in your project folder.
3. Include it:
   ```cpp
   #include "genzcpp.h"
   ```

## Quick Start

Here is a sample Hello World:

```cpp
#include "genzcpp.h"

lock_in
    yap("Hello World.");

    squad<int> nums = {1, 2, 3, 4, 5};

    // Loop 5 times
    spam(5, yap("We spamming"));

    lowkey (nums.size() > 0) {
        yap("We got data.");
    } anyways {
        yap("It's empty.");
    }

    bet
```
